# 🍬 Nassau Candy Profitability Dashboard

## 📌 Project Overview

An interactive Streamlit dashboard developed to analyze the profitability of Nassau Candy products. The dashboard enables users to explore sales, profit, gross margin, cost analysis, and product performance using dynamic filters and business-focused visualizations.

## 🎯 Project Objective

The objective of this project is to help business stakeholders identify:

- Top-performing products
- Low-margin products
- High-cost products
- Profit concentration across products
- Division-wise performance
- Business opportunities for improving profitability

## 🛠️ Technologies Used

- Python
- Pandas
- Streamlit
- Plotly Express
- Plotly Graph Objects
